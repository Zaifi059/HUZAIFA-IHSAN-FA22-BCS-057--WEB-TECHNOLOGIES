‚úÖ SOLUTION 2: Find Your Correct IP
-----------------------------------
The IP shown might be wrong. To find your correct IP:

1. Open Command Prompt (Win + R, type: cmd)
2. Type: ipconfig
3. Look for "IPv4 Address" under your WiFi adapter
4. Use that IP address!

Example: 
   IPv4 Address: 192.168.1.45
   Use: http://192.168.1.45:8000/
