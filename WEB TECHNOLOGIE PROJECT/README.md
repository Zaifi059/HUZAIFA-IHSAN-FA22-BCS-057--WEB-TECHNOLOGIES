## Getting Started

Simply open `index.html` in your web browser to view the website.

### Uploading Images

1. **Portfolio Images**: Place your project images in the `images/` folder
2. **Contact Form**: Use the file upload in the contact section to upload documents/images
3. **File Types**: Supports images (JPG, PNG, GIF) and documents (PDF, DOC, DOCX)



## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
}
```

### Content
Edit the HTML in `index.html` to change text, links, and content.

### Animations
Modify animations in `script.js` to adjust scroll effects and interactions.
