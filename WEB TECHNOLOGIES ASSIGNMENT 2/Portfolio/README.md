# Huzaifa Ihsan - Personal Portfolio Website

A modern, responsive personal portfolio website for Huzaifa Ihsan, Computer Science student (FA22-BCS-057) and owner of URSOFT software house. Built with HTML5, CSS3, and JavaScript.

## Features

### Structure (HTML5)
- ✅ Semantic HTML5 structure with header, nav, main, footer, sections
- ✅ Home section with introduction and welcome message
- ✅ About Me section with biography, education, and interests
- ✅ Skills section showcasing technical and soft skills
- ✅ Projects section with 3 sample projects
- ✅ Contact section with functional form

### Styling (CSS3)
- ✅ External CSS file with consistent theme
- ✅ Modern color scheme with CSS custom properties
- ✅ Flexbox and CSS Grid layouts
- ✅ Fully responsive design (mobile-first approach)
- ✅ Hover effects, transitions, and animations
- ✅ Modern typography with Google Fonts

### Interactivity (JavaScript)
- ✅ Form validation with real-time error messages
- ✅ "Back to top" button with smooth scrolling
- ✅ Mobile navigation toggle (hamburger menu)
- ✅ Smooth scrolling navigation
- ✅ Active section highlighting
- ✅ Typing animation effect
- ✅ Intersection Observer animations
- ✅ Notification system for form submissions

## Technologies Used

- **HTML5**: Semantic markup, accessibility features
- **CSS3**: Flexbox, Grid, Custom Properties, Animations
- **JavaScript (ES6+)**: Modern JavaScript features, DOM manipulation
- **Font Awesome**: Icons
- **Google Fonts**: Typography (Poppins)
