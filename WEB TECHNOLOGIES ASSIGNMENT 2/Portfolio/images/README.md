# Images Folder

This folder contains all the images used in the portfolio website.

## Required Images

### Profile Image
- **File**: `profile.jpg`
- **Size**: 400x400 pixels (recommended)
- **Format**: JPG, PNG, or WebP
- **Usage**: Main profile photo in the home section
- **Description**: Professional headshot of Huzaifa Ihsan

### Project Images
- **File**: `project1.jpg` - URSOFT Business Management System
- **File**: `project2.jpg` - Client Project Management Portal  
- **File**: `project3.jpg` - URSOFT Analytics Dashboard
- **Size**: 600x400 pixels (recommended)
- **Format**: JPG, PNG, or WebP
- **Usage**: Project screenshots in the projects section

## How to Add Your Images

1. **Replace Placeholder Files**: 
   - Replace the placeholder files with your actual images
   - Keep the same file names for the website to work properly

2. **Image Optimization**:
   - Compress images for faster loading
   - Use modern formats like WebP when possible
   - Maintain good quality while keeping file size reasonable

3. **Recommended Tools**:
   - Online image compressors (TinyPNG, Compressor.io)
   - Image editing software (Photoshop, GIMP)
   - Online image resizers

## Fallback

If images are not found, the website will show:
- Gradient placeholder with icons for profile image
- Default project placeholders with Font Awesome icons
